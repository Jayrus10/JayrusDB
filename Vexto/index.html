⚠️ Aviso Importante

📱 Datos temporales

Los datos se almacenan localmente en este navegador. Si borras el caché, se perderán.

💾 Exporta regularmente

Exporta tus datos al menos una vez al día para evitar pérdidas.

🔄 Limpieza

Los datos no se sincronizan entre dispositivos.

👤 Privado

Cada navegador tiene su propia base de datos independiente.

ℹ️ Versión de prueba

No es recomendado empezar a usar la aplicación en su estado actual (beta).
